Privacy Policy

NSN doesn't collect any personal data about its users. It has a read-only database which contains the information about standards, and there is nowhere to store any user information. This means that even if anyone asked to see data about you, we wouldn't have anything to show.

The closest thing to personal data is available in the analytics page. It allows me to see how my server is responding to user input, and how people are accessing this website (e.g. from Google Search results). A user's IP address is logged by the server, but this log is discarded after 90 days, and there is no way for anyone other than myself to access it, and I will never give it to anyone else.

Google Search does keep track of what people search in order to land on my website, and other similar statistics (number of clicks/impressions, average position in search results), and this information is available to me, but nobody else. I only use this information to improve how useful the website is, for example seeing what most people use on this website, whether that's resource access, shared links, credit calculation, or anything else.
